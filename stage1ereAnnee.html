Mes Missions Pendant le Stage

Au cours de mon stage chez Veolia, j'ai eu la chance de contribuer à un projet SaaS innovant. Mes missions principales étaient :

  • Développement de CRUD en Python : J'ai conçu et implémenté des opérations CRUD (Create, Read, Update, Delete) pour plusieurs classes Python. Ces fonctionnalités ont permis de gérer efficacement les données au sein de l'application SaaS.
  • Utilisation de FastAPI pour les tests : Pour garantir le bon fonctionnement des CRUD développés, j'ai utilisé FastAPI. Cet outil m'a permis de tester les différentes opérations de manière rapide et précise, assurant ainsi une performance optimale de l'API.
  • Automatisation de l'enregistrement de fichiers avec AWS : J'ai mis en place des solutions d'automatisation pour l'enregistrement et la gestion des fichiers en utilisant Amazon Web Services (AWS). Cette automatisation avait pour but d'enregistrer un fichier dans des dossiers en fonction de la date et de l'id du fichier.

Outils Utilisés Pendant le Stage

Visual Studio Code

Visual Studio Code est un éditeur de code source développé par Microsoft. Pendant mon stage, je l'ai utilisé comme IDE principal pour écrire et déboguer mon code Python.

Python

Python est un langage de programmation polyvalent et largement utilisé. Au cours de mon stage, j'ai principalement utilisé Python pour développer des CRUD (Create, Read, Update, Delete) dans le cadre du projet SaaS.

FastAPI

FastAPI est un framework moderne, rapide et performant pour créer des API en Python. Pendant mon stage, j'ai utilisé FastAPI pour tester et déployer les CRUD que j'avais développés. FastApi est facile à manipuler et permet de tester les CRUD en python facilement.

AWS

Amazon Web Services (AWS) est une plateforme de services cloud qui offre diverses solutions pour le stockage, la gestion des données, et bien plus encore. J'ai utilisé AWS pour automatiser l'enregistrement et le stockage des fichiers dans le cadre de notre projet SaaS. AWS a simplifié la gestion des ressources.

DynamoDB

DynamoDB est une base de données NoSQL entièrement gérée par AWS, offrant une haute performance et une évolutivité automatique. J'ai utilisé DynamoDB pour stocker et gérer les données générées par les CRUD.
