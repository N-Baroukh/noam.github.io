Mes Missions Pendant le Stage

Au cours de mon stage chez Veolia, j'ai eu la chance de contribuer à un projet SaaS innovant. Mes missions principales étaient :

  • Développement de CRUD en Python : J'ai conçu et implémenté des opérations CRUD (Create, Read, Update, Delete) pour plusieurs classes Python. Ces fonctionnalités ont permis de gérer efficacement les données au sein de l'application SaaS.
  • Utilisation de FastAPI pour les tests : Pour garantir le bon fonctionnement des CRUD développés, j'ai utilisé FastAPI. Cet outil m'a permis de tester les différentes opérations de manière rapide et précise, assurant ainsi une performance optimale de l'API.
  • Automatisation de l'enregistrement de fichiers avec AWS : J'ai mis en place des solutions d'automatisation pour l'enregistrement et la gestion des fichiers en utilisant Amazon Web Services (AWS). Cette automatisation avait pour but d'enregistrer un fichier dans des dossiers en fonction de la date et de l'id du fichier.

Outils Utilisés Pendant le Stage

Visual Studio Code

Visual Studio Code est un éditeur de code source développé par Microsoft. Pendant mon stage, je l'ai utilisé comme IDE principal pour écrire et déboguer mon code Python.

Python

Python est un langage de programmation polyvalent et largement utilisé. Au cours de mon stage, j'ai principalement utilisé Python pour développer des CRUD (Create, Read, Update, Delete) dans le cadre du projet SaaS.

FastAPI

FastAPI est un framework moderne, rapide et performant pour créer des API en Python. Pendant mon stage, j'ai utilisé FastAPI pour tester et déployer les CRUD que j'avais développés. FastApi est facile à manipuler et permet de tester les CRUD en python facilement.

AWS

Amazon Web Services (AWS) est une plateforme de services cloud qui offre diverses solutions pour le stockage, la gestion des données, et bien plus encore. J'ai utilisé AWS pour automatiser l'enregistrement et le stockage des fichiers dans le cadre de notre projet SaaS. AWS a simplifié la gestion des ressources.

DynamoDB

DynamoDB est une base de données NoSQL entièrement gérée par AWS, offrant une haute performance et une évolutivité automatique. J'ai utilisé DynamoDB pour stocker et gérer les données générées par les CRUD.

Compétences Acquises Durant le Stage

Développement avec Python

J'ai renforcé mes compétences en Python, notamment en écrivant et en optimisant des scripts pour les opérations CRUD. J'ai aussi approfondi ma compréhension des bonnes pratiques en matière de structuration de code, de manipulation des données.

Création d'API avec FastAPI

L'utilisation de FastAPI m'a permis d'acquérir des compétences en création d'API rapides et performantes. J'ai appris à gérer les requêtes HTTP, et à tester mon CRUD efficacement.

Gestion des Infrastructures avec AWS

Travailler avec AWS m'a permis d'apprendre le monde du cloud. J'ai appris à automatiser l'enregistrement des fichiers, à configurer des services cloud.

Utilisation de DynamoDB

Grâce à DynamoDB, j'ai acquis des compétences en gestion de bases de données NoSQL. J'ai appris à structurer les données pour des opérations CRUD rapides.

Difficultés Rencontrées

Compréhension des besoins du projet

Au début de mon stage, j'ai eu du mal à comprendre les besoins et les objectifs du projet SaaS. J'ai dû passer du temps à discuter avec l'équipe pour clarifier les attentes et les exigences du projet.

Utilisation d'un nouvel outil

Travailler avec de nouveaux outils de développement comme FastAPI et AWS a demandé un temps d'adaptation. J'ai dû apprendre à les utiliser et à comprendre leur structure pour les intégrer correctement dans le projet.

Gestion des erreurs et débogage

Gérer les erreurs et déboguer le code a été un défi durant mon stage. J'ai souvent rencontré des problèmes inattendus qui ont nécessité de les analyser pour comprendre d'où ils venaient.

Conclusion

Ce stage chez Veolia m’a permis de développer des compétences techniques, partant de la création d’API avec FastAPI à la gestion de base de données avec DynamoDB sur AWS. Le développement de CRUD en Python m'a permis d'avancer le projet SaaS by Veolia. Ce stage m’a également donner l'opportunité de mieux comprendre les bonnes pratiques en matière de développement.

Finalement, cette expérience m'a permis de gagner en compétences techniques de développement et en cloud, ainsi que d'optimiser la gestion de projet
